# Chess Game

### Chess game built using

- [React](https://reactjs.org/)
- [RxJS](https://rxjs-dev.firebaseapp.com/guide/overview)
- [React Dnd](https://react-dnd.github.io/react-dnd/)

### To start

- Clone the repository
- `npm install`
- `npm start`
- Open http://localhost:3000
